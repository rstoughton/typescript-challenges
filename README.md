## Instructions

Solve each of the challenges in `index.ts` to get each of the tests suites in `index.test.ts` to pass.

## Before you get started

```
npm i
```

## Running tests

```
npm test
```

or

```
npm run test:watch
```

## Running the app

In case this helps you with debugging or something

```
npm run dev
```

## Solutions

```
git checkout solutions
```
